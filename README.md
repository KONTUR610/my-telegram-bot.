# Бот-нагадування для адвоката

Цей бот автоматично надсилає повідомлення про початок робочого дня.

## Налаштування для Render:
- **Runtime:** Python
- **Build Command:** `pip install -r requirements.txt`
- **Start Command:** `python main.py`
- **Python Version:** 3.11.0 (вказати в Environment Variables)

## Файли:
- `main.py` — основний код бота.
- `requirements.txt` — список бібліотек.
